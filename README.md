SkillSync is a React Native App that allows teachers, professors, and instructors to create balanced student groups based on personality traits, skills, and learning styles. Instead of random grouping, SkillSync uses a customizable weighted algorithm to generate more effective and intentional teams. 

Features:

-- Creates classroom sessions with a unique 6-digit class pin

-- Students join using code and username of their choice

-- Short Personality and skill-based quiz

-- Customizable trait weights for grouping algorithm

-- Adjustable group sizes

-- Automatic group generation

-- Real-time updates using Firebase

Setup Instructions:

-- Clone the repository

-- Install dependencies
```bash
npm install
```

-- Start the app
```bash
npx expo start
```
